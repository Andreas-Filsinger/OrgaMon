Messe Modus

 Für die Präsentation auf Messen wird ein besonderer Modus von Contutto
 programmiert, nur 800x600 Modus. Im Messemodus startet die Anwendung
 gleich in den (zuletzt ge-wählten) Datenbank-Bereich durch. Alles ist
 rein über die Tastatur bedienbar. Diverse Funktions-Tasten ermöglichen
 die Bedienung rein mit der Tastatur! Start der Anwendung durch:

 ConTutto.exe messe

 Die Bedienung ist vollständig durch die Tastatur möglich

 f5 - play next (spiele ersten/nächsten Sub-Titel zu diesem Stück)
 f6 - stop musik
 f7 - leiser
 f8 - lauter
 f9 - vorhergehender gefundener
 f10 - nächster gefundener
 f11 - Info über den Titel
 f12 - Trefferliste anzeigen
 <esc> offene fenster schliessen

     TITEL.CSV
     =========

     1) "Cdnr"         : Eindeutige Nummer
     2) "CD Titel"     : Name der CD
     3) "Titel"        : Einzeltitel
     4) "Komponist"    : Eingabe = Nachname, Vorname
     5) "Arrangeur"    : Eingabe = Nachname, Vorname
     6) "Dirigent"     : Eingabe = Nachname, Vorname
     7) "Orchester"    : Name Orchester
     8) "Laufzeit"     : Uhrzeitfeld (Laufzeit Einzeltitel)


     CD_DAT1.CSV
     ===========

     1) "CD Titel"          : Name der CD
     2) "Cdnr"              : Eindeutige Nummer
     3) "Land"              : Land
     4) "Internet"          : In diesem Feld stehen Labelname und Bestellnummer als Infofeld
     5) "Kommentar"         : Beschreibung zu einzelnen CDs durch die Redaktion
     6) "Cover1"            : Bildname des dazugehörigen CD-Covers
     7) "Inhaltskuerzel"    : Rubriken als Kürzel
     8) "Lieferant"         : Hier steht unser Lieferant für den Shop. Bei alter CD-ROM war das so gelöst, wenn hier etwas drinsteht, dann konnte der Kunde die CD über uns bestellen.
     9) "Einzelpreis"       : Momentaner Preis der CD bei uns im Shop
     10) "BestellNr."       : Bestellnummer noch einmal separat zum Feld LabelfürInternet. Ist nur gefüllt, wenn wir diese CD im Shop haben.
     11) "Anzahl"           : Wenn es sich um eine Mehrfach-CD handelt, dann ist hier die Anzahl der CDs eingetragen.
     12) "Neuheit"          : Ist hier eine 1 eingetragen, dann handelt es sich um eine Neuheit
     13) "Erstellungsdatum" : Erstellungsdatum des Datensatzes
     14) "Eins"             : Verknüpfung mit Labeldatenbank (Kürzel in Feld "Eins" ist die Verknüpfung)
     15) "EURO"             : Preis umgerechnet in Euro
     16) "Summe_Laufzeit"   : Gesamtlaufzeit der CD

     OUT.CSV
     =======

     1) .. 8) aus TITEL.CSV
     1) .. 16) aus CD_DAT1.CSV

 Datenimport DVO (Beschreibung, aktualisierte Version)
 =====================================================

 Die Erzeugung der Obermayer-Datenbank kann nun selbst erfolgen! Das Import
 Modul ist in Contutto integriert. So kann immer mit den aktuellsten daten
 gearbeitet werden.

 1) Im Filemaker alle Datenfelder exportieren in der reihenfolge, wie
    sie angeboten werden. Es muss nach C:\programme\contutto exportiert
    (bzw kopiert) werden. Es müssen 2 Datenbanken exportiert werden:

    cd_dat1.csv
    titel.csv

    Datei->Import/Export->Daten exportieren->Dateityp (Text mit Komma)->
    Zu weiteren Einstellungen bitte cd_dat1.gif und titel.gif beachten (
    Alle in .\infos Ordner!).

    Vom Contutto-Import Modul werden diese beiden beiden Dateien zu einer
    Zusammengeführt. cd_dat1.csv und titel.csv sind beispielhaft im
    .\infos Verzeichnis gespeichert!
 2) Contutto starten, in den hebu-Bereich _>internet->"DVO 1"->
    die Datei out.csv wird neu erzeugt (Summe der cd_dat1.csv und titel.csv)
    das Fenster schließt sich selbststätig. Nun "DVO 2"
    der Import läuft nun, danach findet man in
    C:\programme\contutto 4 neue Datein, diese folgendermaßen kopieren
    N artikel1.bin kopieren nach .\system\artikel1.bin
    N Texte1.BLA kopieren nach .\system\Texte1.BLA
    N WSI1.bin kopieren nach .\system\WSI1.bin
    N WSS1.bin kopieren nach .\system\WSS1.bin
    (jeweils das N wegnehmen->alte daten überschreiben)
 3) die berühmenten 2 deinstallations-Häkchen, danach neu starten.
    Ab sofort wird mit der neuen Datenbank gearbeitet.
